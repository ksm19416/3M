# Page Turing Over System For Musicians
## Brief
MATLAB을 이용한 악보를 자동으로 넘겨주는 프로그램입니다.<br>
Microphone으로 인식한 연주와 image 형태로 저장된 악보를 대조하여 자동으로 다음 페이지로 넘어갑니다.<br>
